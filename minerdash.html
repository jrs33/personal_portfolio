Miner Dash

        
A sophisticated OOP-based game focused on using miners to get gold while avoiding attacking soldiers

        
Description


This was a semester long project devoted to creating a game that employed all foundational OOP techniques. The game consists of Miners who would go to Gold Mines to collect gold to bring back to towns, while simultaneously avoiding attacking Soldiers who would try to prevent them from completing this task. The entire game was encapsulated in a Model object, which allowed the game to be easily controlled via user input. There were also additional features, such as persistence and an ASCII-grid UI. It was entirely user driven, and was a really fun game to make that reinforced a lot of concepts. 
 


Role


This was a completely independent project, but I was able to gain a lot of mentorship from seniors at BU for help, many of whom I look up to greatly and have positions at companies such as VMWare, Optum, Microsoft, etc. Definitely a fantastic learning experience.
