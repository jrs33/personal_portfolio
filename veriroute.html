VeriRoute

        
Utilizing private insurance data and Google Maps API to provide a statistically proven safe route

        
Description


My first Hackathon as a fresh transfer from Biomedical Engineering to Computer Engineering was a fantastic experience. As the newcomer to this culture, I was able to work in a team to utilize insurance data and Google Maps API to create a web app that provides drivers with the statistically safest route to their destination. The data in question took randomly sampled data describing at what lattitudes and longitudes drivers were speeding, accelerating or texting, upon which we ascribed a scoring system to denote how 'safe' each route would be to drive.
 


Role


I worked entirely on the backend, and learned a ton about handling big data sets, manipulating them, and efficiently displaying computation in a nice interface. Specifically, I worked on integrating the API into the program, and also worked with the scoring system and data manipulation on the back end.
  



Tools Used


Javascript, HTML, CSS
